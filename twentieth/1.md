# JSON语法



## 简单值

```javascript
5
"hello world"
```

js字符串和json字符串最大的区别是：json字符串必须使用双引号



## 对象

```javascript
{
  "name": "Tom",
   "age": 29,
   "school": {
     "name": "SDUT",
     "location": "shandong"
   }
}
```
